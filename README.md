##  Featured Projects

### [Japan PR Point Calculator](https://github.com/rbnhd/pr-point-calc-japan)
A web-based calculator to help highly-skilled foreign professionals assess their eligibility for permanent residency in Japan through the points-based immigration system.

**Technologies:** HTML, JavaScript, CSS
**Live Demo:** [rbnhd.github.io/pr-point-calc-japan](https://rbnhd.github.io/pr-point-calc-japan)

### [JLPT N2 Reading Practice](https://github.com/rbnhd/jlpt-n2-reading)
A comprehensive static website for Japanese language learners preparing for JLPT N2. Features 15 reading passages, 1,600 vocabulary entries, and 100 grammar patterns with local progress tracking.

**Technologies:** HTML5, JavaScript (ES6), CSS3
**Live Demo:** [rbnhd.github.io/jlpt-n2-reading](https://rbnhd.github.io/jlpt-n2-reading)

##  Development

This is a static site built with vanilla HTML, CSS, and JavaScript. No build process required.

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/rbnhd/utils.git
cd utils
```

2. Open `index.html` in your browser or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js http-server
npx http-server
```

3. Visit `http://localhost:8000` in your browser
